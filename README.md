# speedo-generator-hud

## Usage:
- Configure speedos in `speedo/speedo_config.res` and `materials/vgui/replay/thumbnails/speedo/speedo_config.vmt`
- Drag and drop the contents of `speedo-generator-hud-vx.x.x` into your HUD\'s folder. eg: `tf/custom/m0rehud/`
- Add the following line to the top of your HUD\'s `resource/ui/hudplayerclass.res`
  - `#base ../../speedo/speedo.res`
